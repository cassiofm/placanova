⚠️ Navegador incompatível.
Esta página funciona apenas no Google Chrome ou Microsoft Edge. Por favor abra um desses navegadores e acesse o link novamente.

Antes de começar, verifique

🌐 Navegador: Google Chrome ou Microsoft Edge
🔌 Cabo USB conectado entre o computador e o dispositivo
⚡ O dispositivo deve estar ligado (LED aceso)
🔒 Feche qualquer outro programa que possa estar usando a porta USB do dispositivo
Siga os passos abaixo
1
Conecte o dispositivo ao computador

Use o cabo USB fornecido e conecte o dispositivo a uma porta USB do computador. Aguarde alguns segundos até o Windows reconhecê-lo.

2
Clique no botão "Instalar Firmware" abaixo

Uma janela será aberta pelo navegador pedindo para selecionar a porta de comunicação do dispositivo.

3
Selecione a porta do dispositivo

Na janela que abrir, procure e clique na opção que contenha o nome USB Serial ou CH340, depois clique em Conectar.

4
Aguarde a instalação terminar

Uma barra de progresso será exibida. O processo leva aproximadamente 30 a 60 segundos. Não desconecte o cabo durante este tempo.

5
Instalação concluída — desconecte o cabo

Quando aparecer a mensagem "Instalado com sucesso", a atualização foi concluída. Você já pode desconectar o cabo USB. O dispositivo reiniciará automaticamente com o novo firmware.

⚠️ Navegador não suportado.
Use Google Chrome ou Microsoft Edge para continuar.
